# Contributing to Clipp

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repo and clone your fork
2. Set up Supabase credentials (see README)
3. Run `npm run dev` to start the dev server
4. Make your changes in a feature branch

## Code Style

- **CSS**: Use CSS custom properties from `css/variables.css` for colors, fonts, and spacing
- **JavaScript**: Vanilla JS only — no frameworks, no build step
- **HTML**: Use semantic elements (`<section>`, `<article>`, `<nav>`, etc.)
- **Naming**: BEM-inspired class names (e.g., `.feed-card__title`, `.btn--ghost`)

## Pull Requests

1. Create a feature branch: `git checkout -b feature/my-feature`
2. Commit with clear messages
3. Push and open a PR against `main`
4. Describe what changed and why

## Reporting Bugs

Open an issue with:
- Steps to reproduce
- Expected vs actual behavior
- Browser and OS info
